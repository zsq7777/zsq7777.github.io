                                                                                                                                                                                  # 21. C++中的静态

                                                                                                                                                                                  static在C++中有2种分别，根据上下文区分

                                                                                                                                                                                  # 类或结构体外部使用static关键字

                                                                                                                                                                                  类外面的static，以为你申明为static的函数，链接只是在内部，只能对你定义它的翻译单元可见。

                                                                                                                                                                                  # 类或结构体内部使用static关键字

                                                                                                                                                                                  与类的所有实例共享内存，所有该类的实例只有这一个静态变量。 也适用于类中的静态方法，在类中没有实例会传递该方法

                                                                                                                                                                                  # //TODO 静态static在类或结构体范围内的实际含义

                                                                                                                                                                                  # 类和结构体的外部静态

                                                                                                                                                                                  linker不会再这个翻译单元的作用域之外，寻找这个符号的定义

                                                                                                                                                                                  //Static.cpp
static int s_Variable = 5;//该变量只在文件内部链接 ，外部不访问。
                                                                                                                                                                                  1
                                                                                                                                                                                  2
                                                                                                                                                                                  //Main.cpp
#include <iostream> 
int s_Variable = 5; 
 int main() {
	 std::cout << s_Variable << std::endl;
 }
                                                                                                                                                                                  1
                                                                                                                                                                                  2
                                                                                                                                                                                  3
                                                                                                                                                                                  4
                                                                                                                                                                                  5
                                                                                                                                                                                  6

                                                                                                                                                                                  像上面这样运行是没问题

                                                                                                                                                                                  //Static.cpp
static int s_Variable = 5;//该变量只在文件内部链接 ，外部不访问。
                                                                                                                                                                                  1
                                                                                                                                                                                  2

                                                                                                                                                                                  当改成这样： 就会提示链接异常，在其他翻译单元已经定义

                                                                                                                                                                                  # extern 修改变量的实际指向，让他去外部翻译单元寻找

                                                                                                                                                                                  这被称为external linkage或external linking

                                                                                                                                                                                  #include <iostream> 
extern int  s_Variable ; 
 int main() {
	 std::cout << s_Variable << std::endl;
 }
                                                                                                                                                                                  1
                                                                                                                                                                                  2
                                                                                                                                                                                  3
                                                                                                                                                                                  4
                                                                                                                                                                                  5

                                                                                                                                                                                  # 链接器在全局作用域下都不会看到被static修饰的变量，只有它所在的翻译单元可以找到

                                                                                                                                                                                  # 函数被static修饰

                                                                                                                                                                                  static修饰函数时，也意味着它只能被翻译单元内链接，而不能在全局作用域下看到

                                                                                                                                                                                  • 在头文件中使用时，也只是将这个静态变量的代码拷贝到不同的翻译单元。

                                                                                                                                                                                  # 使用

                                                                                                                                                                                  如果不需要全局变量，就尽可能多的使用静态变量，因为一旦在全局作用域申明东西的时候，没有申明static，链接器就会跨编译单元进行链接，这意味已经创建了一个全局的变量。

                                                                                                                                                                                  在任何地方使用，很容易造成bug。所以全局变量要慎用

                                                                                                                                                                                  重点： 要让函数和变量标记为static，除非你真的需要他们跨翻译单元链接

                                                                                                                                                                                  # 总结

                                                                                                                                                                                  在类或结构体的外部使用static修饰变量和函数，会让他们拥有不被外部翻译单元链接的能力。少用全局变量，这可能造成代码混乱。
