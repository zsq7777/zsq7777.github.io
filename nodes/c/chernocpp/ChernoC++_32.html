                                                                                                                                                              # 32. C++字符串

                                                                                                                                                              字符串本质是一个接一个字符组成的一组字符，如字母数字文本 这些都是字符。 一种表示和处理文本的方式。

                                                                                                                                                              # 题外话

                                                                                                                                                              工作原理可以帮助我们更好的理解怎样使用他。

                                                                                                                                                              # 字符是怎样工作的

                                                                                                                                                              C++ 中的char（Character）一个字节的内存。

                                                                                                                                                              # char的作用

                                                                                                                                                              1. 把指针转换成char型指针，所以可以用字节来做指针运算。
                                                                                                                                                              2. 他对于分配内存缓冲区也很有用，如果要分配1K的内存，可以分配1024个char。
                                                                                                                                                              3. 对字符串和文本也很有用，因为C++对待字符默认是通过Ascii字符进行文本编码。

                                                                                                                                                              # 字符编码

                                                                                                                                                              Ascii可以扩展UTF-8 UTF-16 UTF-32

                                                                                                                                                              当然，也有两个字节的字符，3个字节等等 如日文 中文及其他语言。

                                                                                                                                                              1个字节有8byte,也就是2的8次方 256种可能。 字母，数字，符号，中文，日文等 8比特就不够。 所以不同的编码 提供了更多的可能性。

                                                                                                                                                              普通的字符串使用普通的char，我们主要讨论英文字符。

                                                                                                                                                              如游戏种的字符渲染，实际上是一个非常复杂的问题。文本和语言是一个庞大 大规模的问题。

                                                                                                                                                              # 字符串实际上是字符数组，字符数组需要一个空终止字符

                                                                                                                                                              char* name="cherno" 在内存种存在空终止字符，这样我们就知道字符串有多大，从内存地址开始到结束
char name[7]="cherno" 在内存种的表现和char* name="cherno"一样
char name[6]={'c','h','e','r','n','o'} 输出cherno后跟乱码，这实际上是一个数组守卫，让我们知道内存是在我们分配之外。（栈守卫就知道是不是分配在内存之外）
因为std::cout不知道打印到哪里结束，所以输出了一段乱码
char name[7] = { 'c','h','e','r','n','o','\0'};  或者'\0'直接写0，这是在给一个空终止字符，最后输出正常。


const char* name="cherno" 不可更改的常量

                                                                                                                                                              # C++种的标准库 String

                                                                                                                                                              实际上有一个类叫BasicString 他是一个模板类，基本是baseString类的模板版本，模板参数是char。 这叫模板特化 template specialization. 就是将baseString模板类中的模板参数设为char，char是每个字符背后的实际类型。
